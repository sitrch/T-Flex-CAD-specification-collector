# T-FLEX CAD Specification Collector

Инструмент для сбора и экспорта спецификаций из T-FLEX CAD в Excel.

## Описание

Приложение автоматизирует процесс сбора данных из структуры изделия T-FLEX CAD и формирует спецификацию в формате Excel с поддержкой многостраничного экспорта.

## Возможности

- Сбор данных из нескольких плоскостей изделия (Изделия 1-4)
- Экспорт в Excel с разбивкой на листы по разделам
- Поддержка кириллических имен полей и переменных
- Автоматическое обновление структуры изделия перед сбором данных
- Группировка данных по разделам: Детали, Термомосты, Заполнения, Комплектующие, Материалы, Раскрой, Проверка, Другое
- Сохранение истории изменений этажей и количества однотипных этажей

## Структура проекта

- `FragmentSpecificationCollector.cs` — основной класс сбора данных
- `SpecData.cs` — конфигурация изделий и плоскостей
- `SpecificationRow.cs` — модель строки спецификации
- `ExcelExporter.cs` — экспорт в Excel через MiniExcel

## Порядок столбцов в Excel

Плоскость, Этаж, Раздел, Спецификация, Источник, Место установки, Артикул, Артикул базовый, Наименование, Длина, Длина, м, Значение всего, Количество всего, Заполнения тип, Заполнения ширина, Заполнения высота, Заполнения толщина, Заполнения площадь, Этажей
