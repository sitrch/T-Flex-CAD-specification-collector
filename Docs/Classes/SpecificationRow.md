# SpecificationRow

## Назначение
Модель данных для хранения одной записи спецификации, извлеченной из структуры изделия T-FLEX CAD. Содержит все необходимые поля для экспорта в Excel.

## Свойства (столбцы спецификации)

### Основные поля
- `Источник` (string) — имя файла документа (фрагмента), из которого получена запись.
- `Плоскость` (string) — значение из переменной `$Плоскость` корневого документа.
- `Этаж` (int?) — значение из переменной `Этаж` корневого документа.
- `Этажей` (int?) — количество одинаковых этажей из `SpecItemQuantity`.

### Данные спецификации
- `Артикул` (string) — артикул элемента.
- `АртикулБазовый` (string) — базовый артикул.
- `Наименование` (string) — наименование элемента.
- `Спецификация` (string) — признак включения в спецификацию.
- `МестоУстановки` (string) — место установки элемента.

### Числовые параметры
- `Длина` (double?) — длина элемента.
- `ДлинаМ` (double?) — длина в метрах.
- `ЗначениеВсего` (double?) — общее значение.
- `КоличествоВсего` (int?) — общее количество.

### Параметры заполнений
- `ЗаполненияТип` (string) — тип заполнения.
- `ЗаполненияШирина` (double?) — ширина заполнения.
- `ЗаполненияВысота` (double?) — высота заполнения.
- `ЗаполненияТолщина` (double?) — толщина заполнения.
- `ЗаполненияПлощадь` (double?) — площадь заполнения.

### Раздел
- `Раздел` (string) — раздел спецификации (например, "Детали\\Ригели").

## Методы

### `ToExcelRow()`
**Назначение:** Преобразует объект в словарь для записи в Excel через MiniExcel.
**Возвращает:** `Dictionary<string, object>` с ключами-заголовками столбцов.
**Особенность:** Использует `DBNull.Value` для null-значений, чтобы MiniExcel корректно обрабатывал пустые ячейки.

### `GetColumnNames()` (static)
**Назначение:** Возвращает список имен столбцов в порядке, соответствующем `ToExcelRow()`.
**Порядок столбцов:**
1. Плоскость
2. Этаж
3. Раздел
4. Спецификация
5. Источник
6. Место установки
7. Артикул
8. Артикул базовый
9. Наименование
10. Длина
11. Длина, м
12. Значение всего
13. Количество всего
14. Заполнения тип
15. Заполнения ширина
16. Заполнения высота
17. Заполнения толщина
18. Заполнения площадь
19. Этажей

## Использование
Объекты этого класса создаются при обходе структуры изделия в `FragmentSpecificationCollector` и затем сериализуются в Excel.
